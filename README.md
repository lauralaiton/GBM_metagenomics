# GBM metagenomics

## Associated publication
This repository contains the analysis code and workflows used in: **"Laiton, L. and Acevedo, F. E. (2026). Gut microbiome of the grape berry moth, *Paralobesia viteana* (Lepidoptera: Tortricidae) larvae through the grape ripening process revealed by high-throughput 16S and 18S rRNA sequencing. Microbial Genomics, 12(6). https://doi.org/10.1099/mgen.0.001756"**

Raw sequencing data are available through NCBI BioProject PRJNA952655. 
This repository includes the QIIME 2-based 16S and 18S rRNA amplicon analysis pipelines, downstream scripts, and supporting files used to characterize bacterial and fungal communities associated with grape berry moth larval guts from immature and mature Concord grapes.
